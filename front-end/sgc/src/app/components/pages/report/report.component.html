Nessa disciplina podemos ver um escopo maior da área de front-end, alguns frameworks mais utilizados pelos profissionais atualmente. Começamos entendendo um pouco dos conceitos conceitos importantes do front, como mobile first, boas práticas, além de tendências de mercado e como essa área muda e a cada ano surge uma nova tecologia. Também fizemos uma atividade interdisciplinar, que é o projeto favorite, aplicando as boas práticas, e utilizando o que aprendemos até aqui para desenvolver um projeto responsivo e com uma identidade visual.
